# Campus Events — Desktop (teammate-style) Figma import

11 desktop artboards (1280×800) that follow the **teammate's original blue/social mobile design**, reflowed for web. Royal-blue primary (`#5661F2`), white cards, photo blocks, social organizer profile with Following/Followers and About · Event · Reviews tabs.

> This is a separate set from `../` (the indigo + teal redesign). Use whichever direction the team picks.

## How to import into Figma
1. Open your Figma file.
2. Drag any `.svg` onto the canvas (each becomes a **Frame** with editable text + vector shapes).
3. Select all → **Tidy up** to arrange.

## Icons
Icons are simple vector shapes (SVG can't carry an icon font). Swap them with the **Iconify** or **Tabler Icons** Figma plugin for crisp versions.

## Photos
Event/cover imagery is flat color blocks standing in for the teammate's photography — drop real images on top in Figma.
